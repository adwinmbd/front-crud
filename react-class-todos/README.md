# React To-do Class CRUD

> React app to demonstrate CRUD operations using classes.

![Home page](todos-crud.png)

## Install:

```
npm install
```

## To run:

```
npm start
```

Access the app on [http://localhost:3000](http://localhost:3000) in the browser.

## To build:

```
npm run build
```
