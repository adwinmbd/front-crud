# React To-do Hooks CRUD

> React app to demonstrate CRUD operations using hooks.

## Install:

```
npm install
```

## To run:

```
npm start
```

Access the app on [http://localhost:3000](http://localhost:3000) in the browser.

## To build:

```
npm run build
```

## To style code:

```
npm run format
```

## To check code format:

```
npm run lint
```
